### 폼 프로퍼티와 메서드

#### 폼과 요소 탐색하기

- document.forms
  - 기명 컬렉션(이름과 순서가 존재)

#### element.form으로 역참조 하기

#### 폼 요소

- input, textarea

```
input.value = 'new value'
input.checked = true
```

- select, option

```
select.options[0].selected = true
select.value = 'new value'
select.selectedIndex = 0
```
